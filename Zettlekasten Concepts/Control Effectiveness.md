# Control Effectiveness

## 1. Simple Explanation (Feynman Technique)
> Control effectiveness is how good we've tried to stop someone from messing up our stuff.
---
## 2. Technical Details
- **Control effectiveness refers to how well our controls have been put in place.**
- This can include if we are using controls properly and how efficient, easy to use and how appropriate they are.
