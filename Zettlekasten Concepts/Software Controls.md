# Software Controls

## 1. Simple Explanation (Feynman Technique)
> Software controls are ways we can stop people from messing with our stuff digitally, like if you put a password on your phone.
---
## 2. Technical Details
- **Software controls are digital security constrictions like passwords, user authentication, etc that we use to prevent software misconduct.**
