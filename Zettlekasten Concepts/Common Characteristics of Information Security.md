# Common Characteristics of Information Security

## 1. Simple Explanation (Feynman Technique)
> There are always going to be common aspects in what we are keeping safe.
---
## 2. Technical Details
- **There are common characteristics among information security**
- Assets are small in physical size but valuable.
- Physical contact is not needed.
- Security breaches can go undetected.
